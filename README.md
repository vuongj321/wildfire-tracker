# Wildfire Tracker

This wildfire tracker can be used to see all the current wildfires using data from the [NASA EONET API](https://api.nasa.gov/) and displaying the locations on a map using [Google Maps](https://mapsplatform.google.com/resources/blog/streamline-the-use-of-the-maps-javascript-api-within-your-react-applications/?_gl=1*rl1sdy*_ga*MTI2Nzc3MzcwNC4xNzUyNTk2ODA3*_ga_NRWSTWS78N*czE3NTY4NTgyNTgkbzQkZzEkdDE3NTY4NTgyNjAkajU4JGwwJGgw).
## What is used
- TypeScript
- HTML
- CSS
- React

## How to run

Install dependencies

```bash
npm install
```

Create .env file in root folder and input access token from Google
```javascript
VITE_GOOGLE_API_KEY="[INSERT YOUR API ACCESS TOKEN HERE]"
```

Run application

```bash
npm run dev
```
